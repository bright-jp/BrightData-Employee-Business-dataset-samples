この **Bright Data Employee Business Dataset Sample** には **2,519件のレコード** が含まれており、**Bright Data API** を使用して抽出されました。

## Data Points

| Field Name | Description |
|------------|-------------|
| `url` | データセットエントリのURLです。 |
| `profile_url` | 個人のLinkedInプロフィールURLです。 |
| `linkedin_num_id` | LinkedInプロフィールの一意の数値識別子です。 |
| `avatar` | LinkedInアカウントに紐づくプロフィール写真のURLです。 |
| `profile_name` | LinkedInプロフィールに記載されている個人のフルネームです。 |
| `certifications` | 個人が取得した認定資格の詳細です。 |
| `profile_location` | LinkedInプロフィールに基づく個人の現在地です。 |
| `profile_connections` | 個人がLinkedIn上で持つコネクション数のおおよその数です。 |
| `profile_country_code` | 個人の所在地の国を表すISOコードです。 |
| `profile_education_full` | 個人の学歴の詳細です。 |
| `profile_last_education` | 個人の直近の学歴（最終学歴）です。 |
| `profile_experience_full` | 個人の職務経験の詳細です。 |
| `profile_last_experience` | 直近の職位および勤務先企業です。 |
| `profile_followers` | 個人がLinkedIn上で持つフォロワー数です。 |
| `profile_linkedin_id` | 個人のLinkedIn ID文字列です。 |
| `profile_current_position` | 個人の現在の職務上のポジションです。 |
| `profile_current_title` | 個人が現在保持している役職名です。 |
| `profile_activity` | 個人がLinkedIn上で行った最近のアクティビティです。 |
| `profile_posts` | 個人がLinkedInに投稿した内容です。 |
| `profile_about` | 個人のLinkedInプロフィールの概要セクションです。 |
| `profile_courses` | 個人が受講したコースです。 |
| `profile_volunteer_experience` | 個人のボランティア活動の経験です。 |
| `profile_languages` | 個人が話す言語です。 |
| `profile_publications` | 個人が執筆した出版物です。 |
| `profile_recommendations` | 個人が受けた、または個人が行った推薦（レコメンデーション）です。 |
| `profile_recommendations_count` | 個人に紐づく推薦（レコメンデーション）の総数です。 |
| `profile_organizations` | 個人が所属している、または過去に所属していた組織です。 |
| `profile_projects` | 個人が取り組んだプロジェクトです。 |
| `company_name` | 個人が現在勤務している、または過去に勤務していた企業名です。 |
| `company_id` | データセット内における企業の一意の識別子です。 |
| `company_linkedin_url` | 企業のLinkedIn URLです。 |
| `company_size` | 従業員数に基づく企業規模です。 |
| `company_country_code` | 企業本社の所在国を表すISOコードです。 |
| `company_description` | 企業の事業内容の説明です。 |
| `company_other_employees` | 同一企業の他の従業員としてLinkedInに掲載されている人数です。 |
| `employees_in_linkedin` | LinkedInに掲載されている当該企業の従業員総数です。 |
| `company_linkedin_followers` | 企業がLinkedIn上で持つフォロワー数です。 |
| `company_locations` | 企業オフィス所在地の一覧です。 |
| `company_founded_year` | 企業の設立年です。 |
| `company_headquarters` | 企業の本社所在地です。 |
| `company_categories` | 企業が事業を行う業界カテゴリです。 |
| `company_logo` | 企業ロゴのURLです。 |
| `company_slogan` | 企業が使用するスローガンまたはタグラインです。 |
| `company_specialties` | 企業の専門分野および得意領域です。 |
| `company_updates` | 企業に関する最近の更新情報またはニュースです。 |
| `company_website` | 企業の公式Webサイトです。 |
| `company_type` | 企業の種別（例：上場、非上場、非営利）です。 |
| `company_clean_domain` | 企業Webサイトのクリーンなドメイン名です。 |
| `company_revenue_usd` | 企業の概算売上（USD）です。 |
| `company_total_funding` | 企業が受けた資金調達総額です。 |
| `company_total_employees` | 企業で働く従業員総数です。 |
| `company_stock_symbol` | 上場企業の場合の株式ティッカーシンボルです。 |
| `company_is_non_profit` | 企業が非営利組織かどうかを示す指標です。 |
| `company_parent_company` | 該当する場合の親会社名です。 |
| `profile_bio_links` | 個人のプロフィール自己紹介（bio）セクションに含まれるリンクです。 |


## Use Cases

### 1. Human Resources Analytics
従業員の学歴、経験、認定資格を分析してスキルギャップや研修ニーズを特定することで、HR戦略を強化します。

### 2. Market Research
業界横断で従業員プロフィールを調査し、競合の動向や労働力分布の理解に役立てることで、市場調査を実施します。

### 3. Business Development
包括的な従業員および企業データを活用して提携機会を発掘し、事業拡大の取り組みを戦略化します。
